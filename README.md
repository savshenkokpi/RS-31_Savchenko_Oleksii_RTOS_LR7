# RS-31_Savchenko_Oleksii_RTOS_LR7
Номер групи: РС-31

Автор: Савченко Олексій

ОПЕРАЦІЙНІ СИСТЕМИ РЕАЛЬНОГО ЧАСУ ТА 32 БІТНІ МК Лабораторна робота 7. Налагодження середовища розробки для програмування 32-розрядних мікроконтролерів з використанням операційної системи реального часу. На прикладі FreeRTOS.

Обладнання: STM32F405, CubeIDE.

Мета: Ознайомитися з принципами роботи операційної системи реального часу FreeRTOS, навчитися створювати та керувати потоками для керування світлодіодами на STM32F411.

Завдання до лабораторної роботи:
Варіант 3. Створити три потоки для керування світлодіодами PD12, PD14, PD15.
Частоти: PD12 — 0.5 Гц (тривалість 500 мс), PD14 — 1 Гц (тривалість 400 мс), PD15 — 3 Гц (тривалість 150 мс).
Кнопка PA0 змінює тривалість імпульсу PD12 від 500 мс до 800 мс.

Функціонал:

LED1 Task: Генерує імпульсний сигнал тривалістю 500 мс на світлодіоді PС10 з частотою 0.5 Гц, за допомогою кнопки PC13 змінює тривалість імпульсу з 500 мс до 800 мс.

LED2 Task: Генерує імпульсний сигнал тривалістю 400 мс на світлодіоді PС11 з частотою 1 Гц.

LED3 Task: Генерує імпульсний сигнал тривалістю 150 мс на світлодіоді PС12 з частотою 3 Гц.

Підключення:

LED1: GND → LED → резистор 220 Ом → PС10

LED2: GND → LED → резистор 220 Ом → PС11

LED3: GND → LED → резистор 220 Ом → PС12

BUT1: GND → Button → резистор 220 Ом → PC13
